このツールでできること

  • ダッシュボード作品の在庫状況・今後の予定・今月の売上・月別売上グラフをひと目で確認できます。
  • 作品・在庫原画やアクセサリーを登録して、いまどこにあるか（在庫・委託中・展示中・売約済・販売済）を管理します。
  • イベント個展・ワークショップ・レッスンの予定と準備物を管理します。参加する人のチェックもここでします。
  • 参加者参加者やお客さまの連絡先とメモを管理します。
  • 売上売上を記録して、月ごとの合計や種類べつの内訳を確認します。

毎日5分の流れ

  1. 開いたら … ダッシュボードの「きょうやること」を見る。終わった作業はチェック（作品が自動で次の段階に進みます）。
  2. 作品ができたら …「＋ 作品を登録」。タイトルだけでOK。詳しいことは「制作」「販売」などのタブで、あとから好きなときに。
  3. 作品が動いたら … 一覧の状態プルダウンをその場で変更（額装が終わったら「きょうやること」でチェックするだけでもOK）。
  4. 売れたら …「売上」タブで「＋ 売上を記録」。作品とつなげると金額が自動で入り、状態も「販売済」になります。
  5. イベントが決まったら …「イベント」タブで登録。当日は「名簿」ボタンで参加者名簿を印刷。

一覧でできるちょい操作（開かなくてOK）

  • 状態の変更 … カードの状態プルダウンをそのまま変更。
  • 価格の変更 … カードの価格をクリック → 入力してEnter。
  • お気に入り … ☆をクリックで★に。
  • 詳細を見る・書き足す … カードのそれ以外の場所をクリックすると編集画面が開きます。

作品の記録を充実させる

編集画面は「基本／制作／販売／リンク／AI」のタブに分かれています。ふだんは「基本」だけで十分です。

  • 制作状況 … アイデア→制作中→乾燥中→額装待ち→撮影待ち→販売準備→販売中→売約→発送待ち→販売済。額装待ち・撮影待ち・販売準備・発送待ちの作品は「きょうやること」に自動で出てきます。
  • 作品番号 … 登録すると AW-2026-001 のような番号が自動でつきます（変更OK・検索OK）。
  • 画像 … 「基本」タブの「作品の画像」で登録。一覧のサムネイル（小さい写真）をクリックすると大きく見られます。
  • タグ …「青、海、癒し」のように読点やカンマで区切って入力。一覧の「タグ」で絞り込めます。
  • 原価と利益 … フォームの「原価と利益」を開いて材料費などを入れると、原価合計と利益を自動計算します。
  • 制作カルテ … 制作日・制作時間・満足度・使用画材などを記録できます。あとで自分の作品分析に使えます。
  • リンク … BASEやホームページ、SNS投稿のURLを保存すると、作品カードからワンクリックで開けます。
  • ★お気に入り … 作品カードの☆を押すと★に。「★のみ」にチェックするとお気に入りだけ表示できます。
  • 状態の履歴 … 状態（在庫→展示中→販売済など）を変えると日付つきで自動記録。編集画面の「記録」で時系列が見られます。
  • AIアシスタント … 編集画面の「AIアシスタント・販売サポート」のボタンを押すと、紹介文やInstagram投稿文の「お願い文」がコピーされます。ChatGPTやClaudeに貼り付けるだけ。
  • 販売ページ用テキスト … BASE掲載用の下書き（タイトル・サイズ・価格・注意事項）をワンクリックでコピーできます。
  • QRコード … 編集画面のリンク欄にURLを入れて「QRコードを表示・印刷」を押すと、BASE・ホームページ・SNSのQRコードが作れます。展示会のキャプションの横に。
  • しぼり込み … タグのほかに、制作年・価格帯でも探せます。検索欄はサイズ（F4 など）にも反応します。

詳細な項目はぜんぶ折りたたみの中にあるので、ふだんはタイトルと価格だけでもOK。あとから少しずつ書き足せます。

印刷・PDFでの共有

  • 作品リスト …「作品・在庫」タブの「印刷・PDF」ボタン。画面で絞り込んだ内容（例：状態＝委託中だけ）がそのまま一覧表になります。委託先への共有や棚卸しに。
  • 参加者名簿 … イベントカードの「名簿」ボタン。
  • どちらも印刷画面で「PDFとして保存」を選ぶと、ファイルとして保存できます。スタッフさんへのLINEやメール送付に使えます。

データの保存とバックアップ

入力したデータは、このパソコンのブラウザの中に自動で保存されます。保存ボタンを押す必要はありません。

  • 週に1回くらい、右上の「バックアップを保存」を押して、できたファイルを iCloud の「Atelier with 管理ツール」フォルダに入れておくと安心です。作品の画像もバックアップに含まれます（そのぶんファイルは大きくなります）。
  • もしデータが消えてしまったら、「バックアップから復元」で最後に保存したバックアップファイルを選べば戻せます。
  • さらに、1日1回自動で「復元ポイント」（過去14日分・画像はのぞく）も作られています。まちがえて消してしまったときは、右上の「復元ポイント」からその日の状態に戻せます。
気をつけること
・データはこのパソコンの、いつも使っているブラウザに保存されています。別のパソコンやスマホでファイルを開いても、データは表示されません（その場合はバックアップファイルで復元すれば見られます）。
・「削除」した作品や記録は元に戻せません。迷ったら削除せず、メモ欄に書き残しておくのがおすすめです。

困ったときは

  • 表示がおかしいときは、まずページを再読み込み（更新）してみてください。
  • 直したいところ・追加したい機能が出てきたら、Claude（クロード）に「AtelierWith運営ノートの○○を直したい」と伝えれば修正できます。
お試し中の変更は、このブラウザの中だけに保存されます。実際のデータとはつながっていません。
